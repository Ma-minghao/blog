# 常用语

## 杂货

- 日本語は話せません 我不会说日语

話 はな

- もう一度お願いします 能麻烦您再说一次吗

- 翻訳ソフトを使ってもいいですか？ 可以用翻译软件吗？

翻訳　ほんやく、　使　つか

- 写真を撮ってもいいですか？ 请问可以拍照吗？

写真 しゃしん、 撮　と

- 写真、いいですか？

## 餐饮

- これをお願いします 我要一份这个

降 お

使　つか
